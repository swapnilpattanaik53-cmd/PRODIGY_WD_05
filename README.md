🌦️ Weather Forecast Web Application

A modern, responsive, and feature-rich Weather Forecast Web Application that provides real-time weather updates, hourly forecasts, and 7-day predictions for any location worldwide using 
a Weather API.

📌 Project Overview :

This project was developed as Task-05 during my Web Development Internship at Prodigy InfoTech.

The Weather Forecast Web Application is a modern web application that fetches live weather data using a Weather API. Users can instantly check the weather for their current location or search for 
any city or country across the world.

The application not only displays the current weather but also provides a detailed **hourly forecast**, **7-day weather prediction**, and important weather highlights such as humidity, wind speed,
pressure, UV index, visibility, sunrise & sunset timings, and "Feels Like" temperature.

To create a more immersive experience, the interface dynamically changes according to live weather conditions. Rain, thunderstorms, sunshine, clouds, fog, and snow trigger unique visual effects and animated backgrounds,
making the application feel realistic and interactive.
